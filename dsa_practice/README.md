# DSA From Scratch - Practical Track (Python)

This track is for learning by coding, not memorizing.

## How To Use This

1. Open one weekly file.
2. Solve functions in the practice file first.
3. Run tests.
4. Compare with solutions only after trying.
5. Repeat weak topics.

Daily target: 60 to 90 minutes.

- 10 min: read concept
- 35 min: code problems
- 10 min: test and debug
- 5 min: notes (what was hard)

## 8-Week Roadmap

Week 1: Arrays, Strings, Two Pointers, Binary Search
Week 2: Hash Maps, Sets, Sliding Window
Week 3: Stack, Queue, Monotonic Stack basics
Week 4: Linked List
Week 5: Recursion and Backtracking
Week 6: Trees (DFS, BFS)
Week 7: Heaps and Greedy
Week 8: Graph basics and revision

## Practice Rule

- First solve with brute force.
- Then optimize.
- Write time and space complexity under each function.
